# 2. 三量关系
什么是三量关系？
如：
	速度、时间、路程
	总额、单价、数量

> [!NOTE]- 例子
> ![](images/Pasted%20image%2020260611114636.png)
>赋值总量450（最小公倍数求）
>算出单价计算器25， 订书机15， 档案盒9
>$\frac{450-(25\times6 + 8\times15)}{9} = 20$
# 3. 为什么赋值可以
赋值量不一定等于真实量，可能相差n倍（n$\in$R）
比如上面三量关系例子：
	赋值450，真实量实际为450n
	最终求单价计算器25n， 订书机15n， 档案盒9n
	都有n最终做除法是消除了
