# 1. 设方程题型
- 比例型：选项不是比例，是具体数据，所以不用赋值
- 一般题目涉及和差乘积关系，找等量关系

> [!NOTE]- 比例举例
> ![](images/Pasted%20image%2020260611155126.png)
> - 简简单单看甲的比例是5倍数BD二选一
>- 5x : 2x
>- 5x-2:2x+2 = 4:3
>- x= 2, 原来甲10
