# 2. 分子分母同大同小

![](https://cdn.nlark.com/yuque/0/2025/png/32465899/1757413565448-19d3f263-2dd8-4252-bae7-adfeb062ffdf.png)

方法1：直接除，不采用，慢，

方法2：同时找两个分数分子间和分母间的倍数，分子倍数大分子大的大，分母倍数大，分母小的大![](https://cdn.nlark.com/yuque/0/2025/png/32465899/1757413842059-52aa365d-4510-4c88-8de7-fde59b5c54b5.png)

原理

分子分母同大同小，比如分子是2倍，分母是3倍

- 都小的分数分子分母同乘2倍
- 分子变得一样，分母小的本来需要乘以3才是一样但是现在乘以2所以还是小
- 分子同分母小的分数一样
