# 2. 同比环比

同比：和去年同时期相比（年份前推）,同比是年为一个周期  
环比：和上一个统计周期相比（上个月份或者上季度）,环比是按相同时间周期

# 5. 去年、昨年、上年

# 6. 百分比和百分点

百分点是专门来描述两个百分数的相对差值，比如，2025和2024年比人口增加了1个百分点

数值上一个百分点 = 1%

增速说明：增速是增长率的简称
