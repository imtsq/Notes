# 2. 词语侧重
## 2.1 考察方式
重点考察词语某个字的辨析
一般是两个词语某个字不同，解法是组词

如：权利，权力，
- 第一个偏向利益个人权利
- 第二个力量，power

> [!NOTE]- 不要想着一个空就选出答案
> ![](images/Pasted%20image%2020260617151414.png)
> 这个例子说明，不要想着一个空就选出答案，一个空其实可以选多个答案
> 1. 第一空只能排除稀奇（奇怪），稀少，稀疏、稀零都可以
> 2. 第二空<font color="red">固定搭配</font>活动状态

## 2.2 积累
火山活动+状态
滋补 + 身体
滋生 + 消极（腐败、细菌）
跃进：比喻快速前进
广袤 + 大地、田野、时空
开阔+眼界
广阔+天地
宽广 + 视野、胸怀


# 3. 固定搭配
## 3.1 瞻前顾后（动词 + 名词、形容词+名词、.......）
怎么个瞻前顾后？
选项给你动词，看后面名词是什么

> [!NOTE]- 例子
> ![](images/Pasted%20image%2020260617160113.png)
>打破是动词，后面的名词垄断，打破垄断
>深刻地 + 改变，不可能是深刻的支出
>娇弱是形容人这里是风声

# 4. 程度词
词义相近，但是程度不一样

注意程度词语，如更、甚至、乃至等
