# 1. 必要条件格式
- 只有......才....
1. 必要条件是重点
2. 必要条件即“只有”和“才”之间的部分

# 2. 对策
## 2.1. 对策标志词：
1. 应该、应当、应、需要、要、必须 + 做法
2. 通过 / 采取……手段 / 途径 / 措施 / 方式 / 方法 / 渠道，才能……
3. 前提、基础、保障
4. 负有……的义务 /……的必由之路 /……的法门之一 /势在必行/要领在于……
## 2.2. 文段特征：
1. 提出问题 + 分析问题 + 解决问题
2. 提出问题 + 解决问题 + 解释说明（意义效果）
3. 对策 + 解释说明

**问题标志词**：
- 不足、低下、不够、不准
- 困境
- 有待加强
- 挑战，瓶颈，软肋，难题，滞后，， 缺陷、风险，等

## 2.3. 反面提对策
如果/倘若/一旦 + （不怎样）+ （就会）不好的结果

不怎样就改成怎样就是做法

## 2.4. 文段无对策，只阐述问题
选项特征：
1. 有针对性、能够解决问题的对策
2. 概括问题(如果选项没对策，选能概况问题的)

## 2.5. 不能选对策
1. 提问是选问题
2. 主题词和对策不相符（典型牛头不对马嘴）比如大人应该少喝酒，但是他将大人改为年轻人
3. 对策和文段不符合，比如文段是不好好学习考不好，然后答案对策是要考好我们要结交朋友
