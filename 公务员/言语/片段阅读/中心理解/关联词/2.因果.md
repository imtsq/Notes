# 原因 + 结论
- 因为......所以
- 由于......因此

结论最重要

# 直接引导结论
这种一般是用原因解释的关联词
- 所以
- 因此
- 因而
- 故而
- 故
- 于是
- 可见
- 看来

变形考法：
- 在这种情况下、对此、可以说、换言之、总之、可以说、简而言之、换句话说、有鉴于此等


- 前面出现多个分类，后文出现**指示代词**“这些” 也是结论

# 核心
结论才是重点，解释都是啰里啰唆，在解释重结论有点像对解释的顺承


# 文段/选项特征
文段结构：
- 结论 + 原因解释
- 原因 + 结论

正确选项：
- 直接选结论
- 解释**结论**的原因（包含了结论的原因要选择）
