# 1. 什么是拆桥？
- 论据和论点没有必然联系（本身就是逻辑不严谨，一个说牛，一个说马）
- 如：论证：小米很帅 论点：所以，小米很高
- 帅不能证明高
- 拆桥不需要否论点或论据，只需要证明论点和论据的桥不成立即可


# 2. 题干特征/什么时候考虑拆桥
- 没有否论点的选项论点与论据
- 论点与论据话题不一致（主要）
- 提问方式为“削弱论证”（不一定正确，先有时候靠否论点）

## 2.1 部分和整体
因为A不能解释B这种简单（因为小米高，所以小米帅），现在一般出整体部分题
<font color="red">论点为整体，论据是部分</font>（部分不能说明部分）

# 总结
削弱论证就两种：
1. 指出论点和论证两者没关系
2. 部分不能说明整体

# 做题步骤

1. 首先看提问
2. 看题干是不是论点和论据话题一致
3. 选项：
	- A不能导致B这种逻辑错误
	- 部分推整体（可能考察量的多少）


整体和部分虚弱，要不然占比少，要不然其他不这样
