# 2. 什么是搭桥

当然要题目问法是加强/前提

### 2.1.3 解题思维
找论点->找论据->去同存异->对比选项


tips：
一般是提问xxx“前提”一般是加强桥，且搭桥基本上只靠这一道题
知道是加强桥，把论点论据不同的关键词选进去就行

# 3. 加强必要条件
![](images/Pasted%20image%2020260610144242.png)
本来一般情况是加强是加强桥，
但是无搭桥情况：
1. 只有论点
2. 论点和论证是统一话题
出现1或者2，且问加强考虑必要条件

什么是必要条件，本来题目充分了，但是确实选项条件就不成立



# 4. 加强之补充论据
![](images/Pasted%20image%2020260610193018.png)
加强论点几乎不可能，但是加强论据还是可能

补充新论据，只需要看论点就可以

加强桥 = 必要条件>解释>举例
