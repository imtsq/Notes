# 1. 四川考情

考的少，但是还是会考

别怂，这种题看着难，多做点，其实考的就那些，不可能让你选不出

# 2. 题型特征

- 给两组以上的对象、与对象的关系
- 让做对象间的匹配（对号入座，谁是谁）

例如：有ABC三人，有3职业老师、护士、医生，用已知条件<mark class="hltr-r">判定身份</mark>
# 3. 方法

不要无脑用代入/假设

## 3.1. 排除法

### 3.1.1. 适用范围：

题目较明确，给的信息是确定的，选项完整
	什么是选项完整，比如给你三个甲是张三、乙是李四、丙王五这种叫完整
	如果是甲是张三这种叫做不完整
### 3.1.2. 做题方法：

1. 读一句排除一句，

2. 快速找到 谁是谁，谁不是谁（排除）‘
总之：通过已知明确信息，排除不正确选项
### 3.1.3. 常见

| 类型          | 方法    |
| ----------- | ----- |
| 题干确定 + 选项完整 | 优先用排除 |
| 涉及大小 /性别比较  | 常考排除  |

## 3.2	代入法

### 3.2.1. 适用范围：

题<mark class="hltr-r">条件不明确</mark>的，如：题目明确有些条件是错的

为什么不能用排除？当题目条件不明确，用错的条件得错的结论
>[!note]- 图片
>![](images/Pasted%20image%2020260523121338.png)

### 3.2.2.做题方法：

-  假设某个选项正确，将选项带入题干看是否符合题意
- 不要题目上来直接代入，只有题目<mark class="hltr-r">条件不明确情况用代入</mark>
- 怎么代：
	1. 直接代选项，挨个排除
	2. 最小假设法（少量特殊题用）
		- <mark class="hltr-r">条件</mark>：每个人两句话，且每句话一对一错
		- 做法：找那种只出现一次的信息

# 5. 推理起点

## 5.1识别
当排除和代入都不行的时候
- 题目问法确定
	- 选项完整----排除法
	- 选项不完整----推理
- 题目问法不确定----代入
- 特殊选项问：可能、不可能----直接用代入法

<mark class="hltr-r">推理起点有确定关系的来推理</mark> 

![](images/Pasted%20image%2020260523215353.png)
这里推理的句子就是 男->女
1. 做题顺序，首先是提问是肯定，选项不完整，用推理
2. D中一定有个女
3. C一个女的，张去A/B，那么王一定在D

> [!NOTE]- 个人有点难的推理起点
> ![](images/Pasted%20image%2020260523222500.png)
> 1. 识别：一定推出，问法确定，但是选项不完整，方法推理
> 2. 张 李完全不同
> 3. 王 赵只有一个
> 4. 李赵去戊
> 5. 只有王去过丙
> 6. 张没去丁和戊
> 7. 结合张没去过丁戊，只有王去过丙，那么张只取甲乙
> 8. 结合推出张去的甲乙，张 李完全不同，李不可能去丙，那么李是丁戊选B
> 为什么觉得难，没从<mark class="hltr-r">只有这个条件推出张没去丙</mark>
