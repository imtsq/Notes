# 1. 四川考情

考的少，但是还是会考

# 2. 题型特征

- 给两组以上的对象、与对象的关系
- 让做对象间的匹配（对号入座，谁是谁）

例如：有ABC三人，有3职业老师、护士、医生，用已知条件<mark class="hltr-r">判定身份</mark>
# 3. 方法

不要无脑用代入/假设

## 3.1. 排除法

### 3.1.1. 适用范围：

题目较明确，给的信息是确定的
### 3.1.2. 做题方法：

1. 读一句排除一句，

2. 快速找到 谁是谁，谁不是谁（排除）
### 3.1.3. 常见

| 类型          | 方法    |
| ----------- | ----- |
| 题干确定 + 选项网站 | 优先用排除 |
| 涉及大小 /性别比较  | 常考排除  |

## 3.2	代入法

适用范围：

题条件不是确定的，如：题目明确有些条件是错的

为什么不能用排除？当题目条件不明确，用错的条件得错的结论

![](https://cdn.nlark.com/yuque/0/2026/png/32465899/1779374470816-a56984ee-3268-4039-bb52-72e84f3d8439.png)

做题方法：

1. 假设选项正确，用选项带入题干看是否符合题意
2. 不是所有都用代入法，有些带入可能是错的

实战

![](https://cdn.nlark.com/yuque/0/2026/png/32465899/1779375798031-ab73ae8c-173e-426c-90c9-adf2e7fe1b28.png)

选A，代入法

![](https://cdn.nlark.com/yuque/0/2026/png/32465899/1779375963696-6c1ec59a-ad2d-4b22-9e7f-35428125488b.png)

1. 第一眼头都大了，题目描述不明确，代入法，
2. 代入A，樊F、钱：凌就不再G排除
3. 代入B，樊E，胡不在G，排除
4. 代入C，樊F、凌不在G，凌只有E，看似和C选项差不多，但是樊F，说明孙推断全错排除
5. D，樊E，胡不在G，胡F和选项一样，孙：也符合、钱也符合，选D

方法二：最小假设法：

特点：每个人每句话，一对一错

那么找那种只出现一次的推，其中：

- 樊在E，这个E出现一次
- 凌在G，这个凌出现一次

任意找一个，假如樊E对，胡不在G，推胡F对，选D
