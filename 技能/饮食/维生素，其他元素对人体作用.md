#### 脂溶性维生素
1. 维生素A

+ 作用：维持正常视力（尤其是暗光视觉）、皮肤和黏膜健康、免疫功能、生殖健康、基因表达。
+ 缺乏症状：夜盲症、干眼症、皮肤干燥粗糙、免疫力下降、儿童生长迟缓。
+ 主要来源：
    - 动物性（视黄醇）：肝脏、鱼肝油、蛋黄、全脂奶。
    - 植物性（β-胡萝卜素，在体内转化）：
        * 蔬菜：胡萝卜、红薯、南瓜、菠菜、芥蓝、西兰花、红柿子椒。
        * 水果：芒果、哈密瓜、杏。

2. 维生素D

+ 作用：促进钙和磷的吸收，维持骨骼和牙齿健康，调节细胞生长，支持神经肌肉和免疫系统功能。
+ 缺乏症状：儿童佝偻病（骨骼软化）、成人骨软化症、骨质疏松症、肌肉无力。
+ 主要来源：
    - 皮肤合成：日光照射是主要来源。
    - 食物：富含脂肪的鱼类（三文鱼、金枪鱼）、鱼肝油、肝脏、蛋黄、强化奶/谷物。
    - 蔬菜：天然蔬菜中含量极低，部分蘑菇（如香菇）经紫外线照射后可产生维生素D2。

3. 维生素E

+ 作用：强效抗氧化剂，保护细胞膜免受自由基损伤，支持免疫功能，促进血管健康。
+ 缺乏症状：罕见，可能导致神经损伤、肌肉无力、视网膜病变、免疫力下降。
+ 主要来源：
    - 植物油：葵花籽油、红花油、小麦胚芽油。
    - 坚果与种子：杏仁、葵花籽、花生、榛子。
    - 蔬菜：菠菜、西兰花、芦笋、羽衣甘蓝。
    - 牛油果。

4. 维生素K

+ 作用：合成凝血因子，帮助血液正常凝固；参与骨骼代谢，调节钙沉积。
+ 缺乏症状：出血倾向增加（如鼻血、牙龈出血、瘀伤）、新生儿出血性疾病。
+ 主要来源：
    - 维生素K1：
        * 蔬菜：绿叶蔬菜是最好来源（羽衣甘蓝、菠菜、西兰花、生菜、卷心菜、莴苣）。
    - 维生素K2：纳豆（发酵大豆）、奶酪、蛋黄、动物肝脏。

#### 水溶性维生素
B族维生素（共8种，常协同作用）

5. 维生素B1（硫胺素）

+ 作用：能量代谢的关键辅酶，尤其对碳水化合物代谢；维持神经、心脏功能。
+ 缺乏症状：脚气病（疲劳、肌肉无力、神经损伤、心力衰竭）、韦尼克-科尔萨科夫综合征（与酒精中毒相关）。
+ 主要来源：全谷物、糙米、猪肉、豆类、葵花籽、豌豆。

6. 维生素B2（核黄素）

+ 作用：参与能量代谢、细胞呼吸；维持皮肤、眼睛和神经系统健康。
+ 缺乏症状：口角炎、唇炎、舌炎（口腔和唇部炎症）、皮肤油腻/皮炎、畏光。
+ 主要来源：奶制品、鸡蛋、动物内脏、瘦肉、蘑菇、杏仁、菠菜。

7. 维生素B3（烟酸）

+ 作用：能量代谢、DNA修复、细胞信号传导；降低胆固醇（药用剂量）。
+ 缺乏症状：糙皮病（“三D症状”：皮炎、腹泻、痴呆）。
+ 主要来源：鸡肉、鱼肉（金枪鱼、三文鱼）、动物肝脏、花生、全谷物、蘑菇、鳄梨。

8. 维生素B5（泛酸）

+ 作用：参与能量代谢、激素和胆固醇的合成。
+ 缺乏症状：罕见，可能导致疲劳、失眠、手脚麻木。
+ 主要来源：广泛存在于食物中，尤其是动物肝脏、蘑菇、鳄梨、全谷物、西兰花。

9. 维生素B6（吡哆醇）

+ 作用：参与氨基酸代谢、血红蛋白合成、神经递质产生（如血清素）、调节血糖。
+ 缺乏症状：贫血、抑郁、 confusion、舌炎、免疫力下降。
+ 主要来源：鸡肉、鱼肉（金枪鱼、三文鱼）、动物肝脏、土豆、香蕉、鹰嘴豆。

10. 维生素B7（生物素）

+ 作用：参与脂肪、碳水化合物和蛋白质的代谢；维持头发、皮肤和指甲健康。
+ 缺乏症状：脱发、皮疹（尤其在眼鼻口周围）、指甲脆弱、神经系统症状。
+ 主要来源：蛋黄、动物肝脏、坚果（尤其是杏仁）、种子、三文鱼、红薯、菜花。

11. 维生素B9（叶酸/叶酸盐）

+ 作用：DNA合成与修复、细胞分裂、红细胞形成；对预防胎儿神经管缺陷至关重要。
+ 缺乏症状：巨幼红细胞性贫血、疲劳、胎儿神经管缺陷、同型半胱氨酸水平升高（增加心血管风险）。
+ 主要来源：
    - 蔬菜：深绿色叶菜（菠菜、羽衣甘蓝、芦笋、西兰花）、豆类（扁豆、鹰嘴豆）。
    - 动物肝脏、强化谷物。

12. 维生素B12（钴胺素）

+ 作用：红细胞形成、DNA合成、神经系统功能维持。
+ 缺乏症状：恶性贫血（巨幼红细胞性贫血）、疲劳、神经损伤（手脚麻木、记忆力下降）、舌炎。
+ 主要来源：几乎只存在于动物性食品中：肉类、鱼类、贝类、家禽、蛋类、奶制品。植物性来源极少，严格素食者需通过强化食品（如营养酵母、强化植物奶）或补充剂获取。

13. 维生素C（抗坏血酸）

+ 作用：强效抗氧化剂；促进胶原蛋白合成（有益于皮肤、骨骼、伤口愈合）；增强铁吸收；支持免疫功能。
+ 缺乏症状：坏血病（牙龈出血、牙齿松动、伤口不愈、关节疼痛、乏力）。
+ 主要来源：
    - 水果：柑橘类（橙子、柚子、柠檬）、猕猴桃、草莓、番石榴。
    - 蔬菜：红柿子椒、西兰花、 Brussels sprouts、菜花、番茄、绿叶蔬菜。

### 第二部分：主要矿物质与微量元素
#### 常量元素
14. 钙

+ 作用：构成骨骼和牙齿的主要成分；参与肌肉收缩、神经信号传导、血液凝固、激素分泌。
+ 缺乏症状：儿童佝偻病、成人骨软化症、骨质疏松症、肌肉痉挛。
+ 主要来源：奶制品、豆腐（用石膏点的）、沙丁鱼/三文鱼（带骨）、深绿色叶菜（羽衣甘蓝、西兰花、芥蓝，但菠菜中的草酸会抑制钙吸收）、强化豆奶/果汁。

15. 镁

+ 作用：参与300多种酶反应，包括能量产生、蛋白质合成、肌肉和神经功能、血糖和血压调节。
+ 缺乏症状：食欲不振、恶心、疲劳、肌肉痉挛/抽搐、心律不齐。
+ 主要来源：坚果（杏仁、腰果）、种子（南瓜籽）、全谷物、豆类、黑巧克力、菠菜、牛油果、香蕉。

16. 钾

+ 作用：维持体液平衡和渗透压；参与神经冲动传导、肌肉收缩（包括心肌）；有助于降低血压。
+ 缺乏症状：肌肉无力、痉挛、疲劳、便秘、心律不齐。
+ 主要来源：香蕉、土豆（尤其是皮）、红薯、菠菜、牛油果、豆类、番茄、椰子水、酸奶、三文鱼。

#### 微量元素
17. 铁

+ 作用：构成血红蛋白的核心，负责血液中氧气的运输；参与能量代谢。
+ 缺乏症状：缺铁性贫血（疲劳、苍白、头晕、呼吸急促、免疫力下降）。
+ 主要来源：
    - 血红素铁（吸收率高）：红肉、动物肝脏、贝类（如蛤蜊）。
    - 非血红素铁（吸收率受其他食物影响）：豆类、扁豆、豆腐、菠菜、羽衣甘蓝、强化谷物。与富含维生素C的食物同食可提高吸收率。

18. 锌

+ 作用：参与众多酶的功能，支持免疫系统、伤口愈合、DNA合成、味觉和嗅觉、生长发育。
+ 缺乏症状：生长迟缓、免疫力下降、伤口愈合慢、味觉减退、脱发、腹泻。
+ 主要来源：牡蛎（含量极高）、红肉、禽肉、螃蟹、龙虾、豆类、坚果、种子、全谷物、强化谷物。

19. 硒

+ 作用：强效抗氧化酶（谷胱甘肽过氧化物酶）的重要组成部分；支持甲状腺激素代谢；增强免疫力。
+ 缺乏症状：罕见，可能与心肌病（克山病）、免疫力下降、甲状腺功能减退有关。
+ 主要来源：巴西坚果（含量极高，1-2颗即可满足日需）、鱼类（金枪鱼、比目鱼）、肉类、鸡蛋、蘑菇。

20. 碘

+ 作用：合成甲状腺激素，调节新陈代谢、生长发育和体温。
+ 缺乏症状：甲状腺肿（颈部肿大）、甲状腺功能减退（疲劳、怕冷、体重增加）、儿童智力发育受损。
+ 主要来源：碘盐、海产品（海带、紫菜、鱼、虾）、奶制品、鸡蛋。

21. 铜

+ 作用：参与铁代谢、能量产生、结缔组织形成、神经系统的维护。
+ 缺乏症状：贫血、白细胞减少、骨质疏松、神经系统问题。
+ 主要来源：动物肝脏、贝类（尤其是牡蛎）、坚果、种子、全谷物、黑巧克力、蘑菇。

22. 锰

+ 作用：参与骨骼形成、氨基酸、胆固醇和碳水化合物代谢的酶反应；抗氧化功能。
+ 缺乏症状：非常罕见，可能影响骨骼健康和葡萄糖代谢。
+ 主要来源：全谷物、坚果、豆类、菠菜、菠萝、黑巧克力。

23. 铬

+ 作用：增强胰岛素的作用，调节血糖和碳水化合物、脂肪及蛋白质的代谢。
+ 缺乏症状：罕见，可能表现为糖耐量受损（血糖升高）。
+ 主要来源：西兰花、葡萄汁、全麦制品、啤酒酵母、肉类、坚果。

24. 钼

+ 作用：几种关键酶（如黄嘌呤氧化酶）的辅助因子，参与氨基酸代谢和尿酸形成。
+ 缺乏症状：极其罕见。
+ 主要来源：豆类、谷物、坚果、动物肝脏、牛奶。
