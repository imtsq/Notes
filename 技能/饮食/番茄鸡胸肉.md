# 原料

+ 鸡胸肉 x1
+ 番茄 x 2
+ 海椒 x 3
+ 食盐一勺
+ 生抽 凭感觉

# 原料价格
鸡胸肉5元

其它大概1元

总计大概6到7元



步骤

1. 鸡胸肉切片（最好冻硬，好切）
    1. 横向水平分成两半
    2. 铅垂线切成薄片

2. 然后鸡胸肉焯水，直到有白色絮状物生成
3. 清洗焯水完成的鸡胸肉，倒入生抽，一勺食盐不要太多，与勺子平齐腌制，（可根据自己口味调，比如放入姜蒜）

4. 海椒切丝，番茄随意

5. 然后倒油（少量，与锅底齐平）
6. 先鸡胸肉翻炒
7. 后海椒
8. 最后番茄
9. 然后一勺盐（根据自己口味
10. 最后少量水煮
11. over
