# <font style="background-color:rgba(255, 255, 255, 0);">摄入蛋白质</font>
<font style="background-color:rgba(255, 255, 255, 0);">正常男性每天摄入蛋白质含量：</font>**<font style="background-color:rgba(255, 255, 255, 0);">0.8克/公斤体重 * 体重（公斤）</font>**

**<font style="background-color:rgba(255, 255, 255, 0);">力量训练者（增肌为主）</font>**<font style="background-color:rgba(255, 255, 255, 0);">：</font>**<font style="background-color:rgba(255, 255, 255, 0);">1.6~2.2克/公斤体重 * 体重（公斤）</font>**

+ 比如体重70公斤男性，每天摄入蛋白质 = 70 * 1.6 = 112 g（这是保底摄入）

一次性摄入蛋白质过度，并不能全部转换肌肉，一些<font style="background-color:rgba(255, 255, 255, 0);">多余部分可能转为能量或脂肪</font>

<font style="background-color:rgba(255, 255, 255, 0);">饮食建议分多次摄入</font>

# 食物蛋白质含量
| 名称 | 含量 | 百分比 |
| --- | --- | --- |
| 鸡胸肉 | 每100g大约含有**21~23克蛋白质** |  23% |
| 黄豆 | **每100g大概35g蛋白质** | 35% |
| 蛋白粉 | **每100g大概60~70g** | 60%~70% |
| 一个鸡蛋大约6~7g蛋白质 | **** |  |


经济角度（同蛋白质含量）： 黄豆 > 鸡胸肉 > 蛋白粉

但是黄豆说的容易雌化，鸡胸肉太撑
