# 预测
中国未来30年 人口会减少，大概是12亿左右

工厂普通工作更多会上机器替代，还有波下岗潮，现在2026难就业只是开始，



住房会回归居住属性，房价大面积下跌导致人们会集中往一线城市跑，导致一线城市房价飙升，

但是其他经济差点的城市房价会持平。为什么不是房子没人买呢，除非是中国打破房子和婚姻的束缚

其实是一个恶心循环，房价贵导致年轻人不想结婚，不想结婚导致房子卖不出去，导致房价下跌，新生儿减少

新生儿减少导致婴儿产品/服务员第一个崩盘，然后会导致产业链的恶化



农村土地会有新格局，年轻人不愿耕种，导致土地会被企业承包，

AI的发展还是单技能，单功能，不会像一个正常人一样，除非攻克脑科学，知道人是怎么思考的，



# 解决措施：
带动就业才是解决根本，老祖宗的温饱思淫欲，有稳定的工作才能有生娃的需求，996本来就是反人类，两个相亲的时间可能都没有，人变成了机器，科技应该服务于人类，而不是服务于资本，促进就业才能改善民生
