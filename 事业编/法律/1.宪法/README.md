# 1. 国家根本制度-根本政治制度
- 国家根本制度： 社会主义制度
- 根本政治制度：人民代表大会制度

在国家层面走社会主义制度，在政治上是人民代表大会制度


# 2. 统治阶级
不要想到统治阶级感觉都是贬义词，我国的统治阶级是人民

# 3. 政体、国体、国家结构行使
国体和政体，国体规定国家权力归谁所有、谁是国家的主人，政体是在政治上应该怎么管，国家结构就地方和中央这种权力怎么划分


# 4. 党派
在野党：和执政党派是对立关系，想把执政党派弄下去（国外）
执政党：当前当政的党派
参政党：中国独有，就是各党派参政，但是不是决定权，参政议政、民主监督、参加政治协商

是正国级，副总理、国务委员都相当于副总理的待遇。
